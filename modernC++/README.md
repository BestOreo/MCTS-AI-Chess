## ReversiChess

### Game rule

https://en.wikipedia.org/wiki/Reversi

### Compile

```
make
```



### Run

#### Play game

```
./mctsgame
```

At the end of each game, hisory record will be saved locally with timestamp as file name. Player could review the game by running with record filename as argument.



#### Review game

```
./mctsgame <timestamp>
```
